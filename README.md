## Быстрый старт

```bash
python -m src.db
```

## Возможности

- Добавление новых мемов в базу
- Поиск по названию, источнику, году и категории
- Редактирование существующих записей
- Удаление мемов из базы
- росмотр полного каталога

## Структура данных

Каждый мем хранится как кортеж из 5 элементов:

```python
MemeRecord = tuple[int, str, str, int, str]
```

## Основные функции

### Backend

| Функция | Описание | Параметры |
|---------|----------|-----------|
| `create_meme()` | Создаёт новый мем | id, name, origin, year, category |
| `select_memes()` | Поиск мемов по фильтрам | все параметры опциональны |
| `update_meme()` | Обновляет данные мема | id (обязательно), остальное опционально |
| `delete_meme()` | Удаляет мем по ID | meme_id |

### Interface

**Функции пользовательского интерфейса:**
- `run()` — главное меню приложения
- `_add_meme()` — диалог добавления мема
- `_find_memes()` — диалог поиска
- `_update_meme()` — диалог обновления
- `_delete_meme()` — диалог удаления
- `_print_memes()` — вывод списка мемов

**Вспомогательные функции ввода:**
- `_read_int()` — ввод целого числа с валидацией
- `_read_optional_int()` — опциональный ввод числа
- `_read_nonempty_string()` — ввод непустой строки
